\# 我的第二篇 Blog



这是我发布的第二篇博客文章。



通过本次实践，我进一步熟悉了 Hugo 博客的基本结构。Hugo 使用 Markdown 文件保存文章内容，文章文件放在 content/posts 目录下。每次新增或修改文章后，只需要将代码提交到 GitHub 仓库，GitHub Actions 就会自动构建网站，并发布到 GitHub Pages。



这说明静态博客不仅便于维护，也能帮助我理解开源工具、版本管理和自动化部署之间的关系。
